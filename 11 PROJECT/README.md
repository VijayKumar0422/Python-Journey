#  Smart AI Personal Assistant

A powerful Python-based command line AI assistant that can answer questions, remember information, manage tasks, generate passwords, save notes, play games, and maintain chat history.

This project is built using Python and demonstrates concepts like File Handling, JSON, Functions, Modules, Exception Handling, and Modular Programming.

---

#  Features

###  AI ChatBot

* Answer predefined questions
* Teach the bot new questions and answers
* Store knowledge permanently in JSON

### 🧮 Calculator

* Addition
* Subtraction
* Multiplication
* Division
* Power
* Modulus

### 📝 Notes Manager

* Add notes
* Show notes
* Delete notes

### ✅ Todo List

* Add tasks
* Show tasks
* Delete tasks

### 🧠 Memory System

* Remember information
* Recall saved information

### 📜 Chat History

* Automatically save conversations
* Separate history file for each user

### 🔐 Password Generator

* Generate strong random passwords

### ☺️ Joke Generator

* Random programming jokes

### 💡 Motivational Quotes

* Random inspirational quotes

### 🎮 Mini Games

* Guess Number Game
* Rock Paper Scissors

### 📅 Date and Time

* Show current date
* Show current time

### 👤 User System

* Register users

### 📊 Statistics

* Total questions

* Unknown questions

#  Available Commands

| Command     | Description         |
| ----------- | ------------------- |
| hello       | Greeting            |
| how are you | Ask bot status      |
| quote       | Random quote        |
| joke        | Random joke         |
| calculator  | Open calculator     |

| add task    | Add task            |
| show task   | Show tasks          |
